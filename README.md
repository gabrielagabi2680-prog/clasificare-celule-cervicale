Clasificarea celulelor cervicale folosind rețele neuronale convoluționale
Lucrare de licenta
Descriere
Sistem de clasificare automată a celulelor cervicale în trei clase clinice conform sistemului Bethesda:

NEGATIV — celule normale
LOW_GRADE — leziuni de grad scăzut (LSIL, ASC-US)
HIGH_GRADE — leziuni de grad înalt (HSIL, ASC-H)

Dataset-uri utilizate

Herlev — 917 imagini, 7 clase
SIPaKMeD — 4049 imagini, 5 clase
CRIC — 11.534 adnotări
CerviAssist — dataset privat adnotat manual
